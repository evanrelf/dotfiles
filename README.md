# dotfiles

Currently I'm using:

- Editor: [Doom Emacs][doom-emacs]
- Shell: [fish][fish]
- Terminal: [kitty][kitty]
- Window Manager: [Amethyst][amethyst] on macOS, [XMonad][xmonad] on NixOS
- Package Manager: [Nix][nix]
- OS: macOS, [NixOS][nixos]
